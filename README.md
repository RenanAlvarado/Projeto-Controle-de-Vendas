# 🛒 Sistema de Controle de Vendas

## 📌 Sobre o projeto
O **Sistema de Controle de Vendas** é uma aplicação desenvolvida com o objetivo de gerenciar produtos, vendas e fluxo básico de um comércio.

O sistema permite realizar o cadastro de produtos, controle de estoque e registro de vendas, facilitando a organização e acompanhamento das operações comerciais.

Este projeto foi desenvolvido com foco em aprendizado prático e aplicação de conceitos de desenvolvimento de software.

---

## 🎯 Objetivos
- Praticar conceitos de desenvolvimento full stack
- Simular um sistema real de controle de vendas
- Aplicar boas práticas de organização de código
- Desenvolver uma aplicação funcional e utilizável

---

## ⚙️ Funcionalidades

- 📦 Cadastro de produtos  
- 📝 Registro de vendas  
- 📊 Controle de estoque  
- 🔍 Consulta de produtos  
- 💰 Cálculo automático de valores
